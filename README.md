## Comandos para rodar o projeto na sua máquina:
(Certifique-se que o Node e Git estejam instalados na sua máquina!)


## 1º
```bash
git clone https://github.com/LucasAlberto0/DesafioToDoList.git
```

## 2º
```bash
cd DesafioToDoList
code .
```

## 3º
```bash
npm install tsc
```

## 4º
```bash
npm install --save-dev typescript ts-loader
```

## 5º 

```bash
npm run dev | em seguinda abrir o index.html no navegador  
```
